📋 ШАГ 1: ПОДГОТОВКА
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Добавьте ваши фотографии в: public/photos/
   (photo1.jpg, photo2.jpg, photo3.jpg...)

2. Добавьте музыку в: public/music/background.mp3

3. Обновите список фото в: src/app/App.tsx


🐙 ШАГ 2: GITHUB
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

А. Создайте аккаунт на https://github.com (если нет)

Б. Создайте новый репозиторий:
   - Нажмите: + → New repository
   - Имя: family-photo-album
   - Public или Private
   - НЕ добавляйте README, .gitignore, license
   - Create repository

В. Откройте терминал в папке проекта (Alt+F12 в PyCharm)

Г. Выполните команды (ЗАМЕНИТЕ ваш-username на свой!):

   git init
   git config --global user.name "Ваше Имя"
   git config --global user.email "your@email.com"
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/ваш-username/family-photo-album.git
   git branch -M main
   git push -u origin main

Д. При запросе авторизации - войдите через GitHub


🌐 ШАГ 3: RENDER.COM
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

А. Перейдите на https://render.com

Б. Sign up with GitHub → разрешите доступ

В. New → Static Site

Г. Найдите репозиторий family-photo-album → Connect

Д. Настройки:
   Name:            family-photo-album
   Branch:          main
   Build Command:   npm install && npm run build
   Publish Directory: dist
   
Е. Create Static Site

Ж. Дождитесь деплоя (2-5 минут)


✅ ГОТОВО!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Ваш сайт будет доступен по адресу:
https://family-photo-album.onrender.com

(или другое имя, которое вы указали)


🔄 ОБНОВЛЕНИЕ САЙТА (в будущем)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Когда захотите добавить новые фото:

1. Добавьте фото в public/photos/
2. Обновите src/app/App.tsx
3. В терминале:

   git add .
   git commit -m "Добавлены новые фото"
   git push

4. Render автоматически обновит сайт!
